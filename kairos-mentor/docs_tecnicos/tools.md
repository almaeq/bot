# Herramientas Disponibles
El Agente tiene permiso para realizar las siguientes acciones en la VM:
1. **crear_archivo(nombre, contenido)**: Crea archivos de código o documentación.
   - Rutas permitidas: `/app/agente/`, `/app/docs_tec/`, `/app/raiz_proyecto/`.
2. **leer_contexto()**: Analiza archivos locales para dar respuestas precisas.
